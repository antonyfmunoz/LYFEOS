Create a fullstack app that functions as a daily dashboard system for a gamified personal development experience (inspired by Obsidian’s daily notes). 

Include the following components:

1. FRONTEND (HTML/CSS/JS):
- A full-screen UI with a dark theme styled dashboard titled “🧠 Game of Lyfe: Daily Dashboard”
- Editable Routine Builder with input fields for:
    - Time Block (e.g., “5:00am - 6:00am”)
    - Block Name (e.g., “Morning Ritual”)
    - Tasks (comma-separated sub-items)
- A "Today’s Routine" display area that pulls in the user-defined routine and formats it as a checklist
- Daily Reflection area with:
    - Sliders or inputs for Mental, Physical, and Emotional State (1–10)
    - Text area for Gratitude & Reflections
- Thought Capture, Content Consumed, and To-Do Idea text areas
- “💾 Save Daily Log” button that writes all data to `localStorage`
- Display current date dynamically
- Display save confirmation when log is stored

2. BACKEND (Vanilla JS localStorage):
- Store:
    - User's custom routine (editable)
    - Daily log for each date (mental/physical/emotional stats, reflections, thoughts)
- Ensure data persists across sessions
- Use localStorage to simulate database
- Include code to retrieve saved data on load

3. JAVASCRIPT LOGIC:
- On page load:
    - Retrieve today’s date
    - Load the user’s routine from storage
    - Load today's log if already written
- On submit:
    - Save full log under date key (e.g. `dailyLog-2025-04-15`)
    - Update routine permanently in localStorage
- Render all components dynamically
- Allow deletion of individual routine blocks

4. USER EXPERIENCE:
- Fully responsive layout with smooth UI transitions
- Styled buttons, inputs, and task lists using modern CSS (dark mode friendly)
- Ready for expansion with XP systems or data visualization (keep code modular)

5. FILE STRUCTURE:
- index.html (markup)
- style.css (dark modern UI)
- app.js (core logic and storage)
- (Optional) data.json for future backend simulation

Ensure the app is clean, minimal, gamified in feel, and built for scaling. It must feel like a digital command center for life execution. Make it feel like the control panel of an elite operator, designed for immersion.

Do not include frameworks like React or Firebase yet — this is an MVP built using only HTML, CSS, and vanilla JS.

Once generated, run the project and display a working UI that users can edit, save, and return to every day.
