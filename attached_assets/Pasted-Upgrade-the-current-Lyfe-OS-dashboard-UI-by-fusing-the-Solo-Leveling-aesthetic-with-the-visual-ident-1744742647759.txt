🎨 STYLE GUIDE: SOLO LEVELING × GAME OF LYFE
–––––––––––––––––––––––––––––––––––

1. 🧠 **Font + Typography**
- Headings: bold, sleek, tech-forward (e.g., Orbitron, Exo 2, or Rajdhani)
- Body text: light and clean sans-serif (Poppins, Inter)
- Color: Use white or silver text with neon-accent glow on headers

2. 🌌 **Color Palette**
- Primary: Deep black, obsidian gray
- Accents: Neon blue, cyber violet, soft electric cyan
- Optional highlights: Magenta, emerald, or golden glow for phase unlocks or XP earned

3. 💠 **Layout + Structure**
- Full-page dashboard (mobile + desktop friendly)
- Soft HUD-style boxes with glowing neon borders
- Sections should feel like RPG panels: Daily Routine, Reflection, XP Tracker, Thought Logs
- XP area should feel like a stat tracker (not cartoonish)
- Use grid layouts, dividers, and sci-fi alignment

4. 🔘 **Buttons & Interactives**
- Glowing borders with soft hover animation
- Checkboxes styled like modern UI toggles or spell seals
- XP check-ins animate with soft glow pulse

5. 📊 **Sliders + Stats**
- Mental / Physical / Emotional sliders should look like RPG stats or energy meters
- Animate smoothly on input change
- Bars glow subtly when filled past 50%

6. 🖼️ **Animations & Effects**
- Fade/slide in sections on page load (1–2s delay per)
- Hover glow on inputs/buttons
- Optional: Add subtle pulse or flicker effect when XP is gained
- Do not overdo motion — cinematic, not noisy
